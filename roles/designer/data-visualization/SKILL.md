# 数据可视化设计

## 核心原则

```
1. 信息传达优先
   - 用户想知道什么？
   - 数据要说明什么？
   - 用户看完应该做什么决策？

2. 数据墨水比最大化
   - 删除不传达信息的元素
   - 减少视觉噪音
```

---

## 图表选择指南

| 数据关系 | 推荐图表 | 适用场景 |
|----------|----------|----------|
| **趋势** | 折线图、面积图 | 时间序列、变化趋势 |
| **对比** | 柱状图、条形图 | 不同类别比较 |
| **占比** | 饼图、环形图、树图 | 部分与整体关系 |
| **分布** | 直方图、箱线图、散点图 | 数据分布情况 |
| **关系** | 散点图、气泡图、热力图 | 变量相关性 |
| **排名** | 条形图、排行榜 | 排序比较 |
| **流程** | 漏斗图、桑基图 | 转化流程 |

---

## 图表设计规范

### 折线图
- 线条数量 ≤ 5 条
- 线条粗细：2-3px
- 重要数据线高亮
- Y 轴从 0 开始（除非有意义）

### 柱状图
- 柱子间距：柱宽的 30-50%
- 按数值大小排序
- 数值标签放柱顶或柱内
- 横向条形图适合长标签

### 饼图/环形图
- 分类 ≤ 5 个
- 按大小排序，最大从 12 点开始
- 环形图中心可放核心指标
- 避免 3D 饼图

## Dashboard 设计模式

### 布局结构

```
标题 + 时间选择器
    │
    ▼
KPI 卡片区（4 个核心指标）
    │
    ▼
主图表 + 次图表
    │
    ▼
明细表格
```

### KPI 卡片

```
┌─────────────────────────┐
│ 📊 指标名称             │
│ ─────────────────────── │
│ 12,345                  │  ← 主数值（24-32px）
│ ─────────────────────── │
│ ↑ 12.5% vs 上周         │  ← 对比/趋势
│ 📈 迷你趋势图           │  ← 可选
└─────────────────────────┘
```

### 时间范围选择器

```
快捷选项：今天 | 7天 | 30天 | 自定义

交互：
- 点击立即生效
- 记住用户选择
- 支持 URL 分享
```

## 交互设计

### Hover 详情
- 延迟 100-200ms 显示
- 不遮挡其他数据
- 包含上下文信息

### 下钻交互
```
图表概览 → 点击数据点 → 显示详情 → 双击展开下级
```

### 筛选联动
```
筛选器 → 图表 A/B/C 自动更新
```

## 颜色系统

### 语义色

| 语义 | 颜色 | 用途 |
|------|------|------|
| 正向 | #10b981 | 增长、成功 |
| 负向 | #ef4444 | 下降、失败 |
| 中性 | #6b7280 | 无变化 |
| 警告 | #f59e0b | 告警 |
| 信息 | #3b82f6 | 提示 |

### 色盲友好

- 颜色 + 形状区分
- 颜色 + 线型区分
- 直接标注文字

## 状态处理

### 加载状态

```
选择其他时间范围或添加数据

```

```

[重试]
```
