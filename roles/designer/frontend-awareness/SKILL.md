# 前端感知

## HTML/CSS 基础
- 了解 Flexbox / Grid 布局能力和限制
- 知道哪些效果 CSS 能实现、哪些需要 JS
- 设计时考虑 DOM 结构的合理性

## 响应式约束
- 知道不同设备的实际渲染差异
- 设计稿标注响应式行为（折叠、隐藏、重排）
- 考虑触摸目标大小（最小 44x44px）

## 性能感知
- 图片：合适格式（WebP）、合适尺寸、懒加载
- 动效：GPU 加速属性（transform, opacity）优先
- 字体：控制字体文件数量和大小

## 可行性驱动
- 设计前跟前端确认技术可行性
- 复杂动效标注实现方式建议
- 不设计"只在 Figma 里好看"的东西
