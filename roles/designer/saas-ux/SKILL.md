# SaaS UX

## Dashboard 设计
- 首屏展示最重要的 3-5 个指标
- 数据有对比（环比、同比），不是孤立数字
- 支持自定义和筛选，不同角色看不同视图

## 复杂数据可视化
- 图表类型匹配数据关系：趋势用折线、占比用饼图、对比用柱状图
- 数据量大时提供聚合和下钻
- 交互：hover 显示详情、点击跳转明细

## 多角色 UX
- Admin 和普通用户的界面分离
- 权限控制在 UI 层体现：无权限的功能隐藏或禁用+提示
- 角色切换体验流畅

## 空状态 / 错误 UX
- 空状态：不是空白页，是引导用户下一步操作
- 加载中：骨架屏优于 spinner
- 错误：说清楚出了什么问题、用户能做什么
- 网络异常：离线提示 + 重试按钮
