# UI 设计

## 视觉层级
- 大小、颜色、间距三者配合建立层级
- 每个页面只有一个视觉焦点
- 次要信息弱化，不是隐藏

## 字体系统
- 字号阶梯：不超过 5 级（12/14/16/20/24 或类似）
- 行高：正文 1.5，标题 1.2-1.3
- 字重：Regular + Medium + Bold 三档够用

## 色彩系统 (Design Tokens)
- 语义化命名：primary, secondary, success, warning, error
- 每个颜色有完整色阶（50-900）
- 暗色模式：不是简单反转，是重新定义语义色

## 栅格与布局
- 12 列栅格，间距 16/24/32
- 内容区最大宽度限制（1200-1440px）
- 组件间距用 4px 倍数

## 响应式
- 断点：mobile (< 768) / tablet (768-1024) / desktop (> 1024)
- 移动优先设计，桌面端增强
- 关键操作在所有尺寸下可达
